# Svelte 시작

## 설치

`yarn berry` 를 사용하기 때문에 yarn 을 최신버전으로 설치해주셔야 합니다.

`zero-install` 을 사용하는 프로젝트로 `yarn install` 을 사용하지 않아도 됩니다.

```cmd
// 설치
npx degit sveltejs/template svelte-study

// 폴더 이동
cd svelte-study

// yarn 설치
npm install -g yarn

// 패키지 설치 (yarn berry 사용), zero-install 을 사용함으로 설치 하지 않아도 됨.
// yarn

// 실행
yarn dev
```
